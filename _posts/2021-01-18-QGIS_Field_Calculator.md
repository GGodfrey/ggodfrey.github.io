The field calculator is a powerful tool for calculating values in the attribute table.

You can create a new field, a virtual field  or update an existing field.

https://docs.qgis.org/2.8/en/docs/user_manual/working_with_vector/field_calculator.html

<a href=""https://gisdriverslicence.files.wordpress.com/2020/09/field-calculator.png""><img class=""alignnone size-medium wp-image-36"" src=""https://gisdriverslicence.files.wordpress.com/2020/09/field-calculator.png?w=300"" alt="""" width=""300"" height=""130"" /></a>

Some basic syntax for expressions is as follows:
<ul>
 	<li>Fields are denoted with speech marks """", e.g. ""FieldA""</li>
 	<li>Text strings are denoted with inverted commas '', e.g. 'Free text'</li>
 	<li>Numbers don't need inverted commas</li>
 	<li>Strings or expressions can be joined or concatenated together using || . There is a shortcut button above the expression box.</li>
</ul>
There is a huge array of functions listed in the middle panel. For more complex calculations it is  possible to write python functions into the functions editor.

https://www.lutraconsulting.co.uk/blog/2015/06/05/qgis-function-editor/
