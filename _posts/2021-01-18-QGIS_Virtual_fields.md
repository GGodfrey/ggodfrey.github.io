Virtual fields allows you to add temporary columns to a table and populate it with an expression. e.g. $length  would show the length. This autogenerates every time you make a new feature or amend an existing feature.

There are some issues though. See this post:https://gis.stackexchange.com/questions/340464/forcing-virtual-field-recalc-in-attribute-table-in-qgis
